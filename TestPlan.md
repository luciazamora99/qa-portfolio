## 1. Objective
The objective of this test plan is to verify that the Gym Workout web application works correctly and provides a consistent and user-friendly experience.

---

## 2. Scope

### In Scope
- Navigation between pages (Inicio and Recetas)
- Opening and closing routine pop-ups
- Opening and closing recipe pop-ups
- Display of workout and recipe content
- Basic UI layout and alignment
- Smooth navigation behavior

### Out of Scope
- Performance testing
- Security testing
- Accessibility testing
- Automated testing

---

## 3. Test Types
- Manual functional testing
- UI testing
- Exploratory testing
- Regression testing (basic)

---

## 4. Test Environment
- Application type: Web application
- Browsers: Edge
- Operating System: Windows 11 Home
- Device: Desktop

---

## 5. Test Approach
Testing will be performed manually by executing predefined test cases and exploratory testing. Any deviation from the expected behavior will be documented as a bug.

---

## 6. Entry Criteria
- Application is deployed and accessible via GitHub Pages
- Main pages load without errors

---

## 7. Exit Criteria
- All planned test cases have been executed
- Identified bugs have been documented
- Test summary report completed

---

## 8. Deliverables
- Test Plan
- Test Cases
- Bug Reports
- Test Summary Report
